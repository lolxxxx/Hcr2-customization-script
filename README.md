# Hcr2-customization-script
A script for gg that can add any customization in-game for hill climb racing 2.

Latest version : v1.2.1
[PATCH NOTES]
* added the beast hat

[FUNCTIONALITY]
- adds skins
- adds paints
- adds animation
To a new/existing account
